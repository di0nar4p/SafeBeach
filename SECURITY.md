# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please **do not** open a public GitHub issue.

Instead, report it responsibly by sending an email to:

📧 **di0nar4p@proton.me**

Please include as much detail as possible:
- A description of the vulnerability
- Steps to reproduce the issue
- Potential impact
- Any suggested fixes (optional)

I will acknowledge your report within **72 hours** and keep you updated on the progress toward a fix.

## Disclosure Policy

- Please give me reasonable time to investigate and fix the issue before any public disclosure.
- I am committed to working with security researchers in good faith.
- Credit will be given to researchers who responsibly disclose vulnerabilities.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | ✅ Yes             |
| older   | ❌ No              |
